# 如何確認和安裝 Node.js

## 確認步驟

### 1. 檢查是否已安裝 Node.js

打開 PowerShell 或命令提示字元（CMD），執行以下命令：

```powershell
node --version
```

如果顯示版本號（例如 `v18.17.0`），表示已安裝。

如果顯示錯誤訊息，表示**尚未安裝 Node.js**。

### 2. 檢查是否已安裝 npm

執行以下命令：

```powershell
npm --version
```

如果顯示版本號（例如 `9.6.7`），表示已安裝。

## 如果尚未安裝 Node.js

### 方法一：從官網下載（推薦）

1. 前往 Node.js 官網：https://nodejs.org/
2. 下載 **LTS 版本**（長期支援版本，較穩定）
3. 執行安裝程式，按照指示完成安裝
4. **重要**：安裝時請勾選 "Add to PATH" 選項
5. 安裝完成後，**重新開啟** PowerShell 或命令提示字元

### 方法二：使用 Chocolatey（如果已安裝）

```powershell
choco install nodejs
```

## 安裝完成後的步驟

### 1. 確認安裝成功

重新開啟 PowerShell，執行：

```powershell
node --version
npm --version
```

兩個命令都應該顯示版本號。

### 2. 安裝專案依賴

在專案目錄中執行：

```powershell
cd c:\Users\tsung\Downloads\glasses-shop
npm install
```

這會安裝所有需要的套件，可能需要幾分鐘時間。

### 3. 啟動開發伺服器

執行：

```powershell
npm run dev
```

應該會看到類似以下的訊息：

```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

### 4. 打開瀏覽器

在瀏覽器中打開：`http://localhost:5173`

## 常見問題

### Q: 安裝 Node.js 後，命令還是找不到？
A: 請**重新開啟** PowerShell 或命令提示字元，讓系統重新載入環境變數。

### Q: npm install 執行很久？
A: 這是正常的，第一次安裝需要下載很多套件，請耐心等待。

### Q: 出現權限錯誤？
A: 可能需要以**系統管理員身份**執行 PowerShell。

### Q: 如何確認開發伺服器正在運行？
A: 如果執行 `npm run dev` 後，終端機顯示了本地網址（如 http://localhost:5173），並且沒有錯誤訊息，就表示正在運行。
