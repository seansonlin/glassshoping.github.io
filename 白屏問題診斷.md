# 白屏問題診斷指南

## 快速檢查步驟

### 1. 確認開發伺服器正在運行
- 終端機應該顯示類似訊息：
  ```
  VITE v5.x.x  ready in xxx ms
  ➜  Local:   http://localhost:5173/
  ```
- 如果沒有，請執行：`npm run dev`

### 2. 檢查瀏覽器控制台（最重要！）
1. 按 `F12` 打開開發者工具
2. 點擊 **Console** 標籤
3. 查看是否有**紅色錯誤訊息**
4. **請複製所有錯誤訊息給我**

### 3. 檢查網路請求
1. 在開發者工具中點擊 **Network** 標籤
2. 重新整理頁面（F5）
3. 查看是否有失敗的請求（紅色）
4. 特別注意：
   - `main.tsx` 是否成功載入？
   - `index.css` 是否成功載入？
   - 其他 `.js` 或 `.css` 檔案是否有錯誤？

### 4. 檢查 HTML 結構
1. 在開發者工具中點擊 **Elements** 或 **檢查** 標籤
2. 查看 `<div id="root">` 裡面是否有內容
   - 如果有內容但看不到 → CSS 問題
   - 如果完全空白 → JavaScript 問題

### 5. 常見錯誤和解決方法

#### 錯誤：`Cannot find module` 或 `Failed to resolve import`
- **原因**：模組路徑錯誤或檔案不存在
- **解決**：檢查所有 import 路徑是否正確

#### 錯誤：`Unexpected token` 或語法錯誤
- **原因**：程式碼語法錯誤
- **解決**：檢查錯誤訊息指出的檔案和行數

#### 錯誤：`React is not defined`
- **原因**：React 未正確導入
- **解決**：確認所有組件都有 `import React from 'react'`

#### 錯誤：`Cannot read property` 或 `undefined`
- **原因**：變數或物件未正確初始化
- **解決**：檢查 Context 或狀態管理是否正確設定

### 6. 測試簡單版本

如果還是有問題，可以暫時使用簡單版本測試：

1. 修改 `src/main.tsx`：
   ```tsx
   import React from 'react'
   import ReactDOM from 'react-dom/client'
   
   ReactDOM.createRoot(document.getElementById('root')!).render(
     <div style={{ padding: '50px', fontFamily: 'Arial' }}>
       <h1 style={{ color: 'blue' }}>測試頁面</h1>
       <p>如果您看到這個，表示 React 基本功能正常。</p>
     </div>
   )
   ```

2. 如果簡單版本可以顯示，問題可能在 App.tsx 或路由設定

3. 如果簡單版本還是白屏，問題可能在：
   - Vite 配置
   - TypeScript 編譯
   - 瀏覽器相容性

## 請提供以下資訊

1. **瀏覽器控制台的完整錯誤訊息**（最重要！）
2. **瀏覽器名稱和版本**（Chrome、Edge、Firefox 等）
3. **終端機的完整輸出**（執行 `npm run dev` 後的訊息）
4. **Network 標籤中是否有失敗的請求**

有了這些資訊，我就能準確找出問題並修復！
