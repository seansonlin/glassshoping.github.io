白屏問題診斷步驟
================

步驟 1: 測試簡單版本
-------------------
1. 暫時將 src/main.tsx 改名為 src/main.original.tsx
2. 將 src/main.test.tsx 複製為 src/main.tsx
3. 重新整理瀏覽器
4. 如果看到測試頁面 → 問題在 App.tsx 或路由
5. 如果還是白屏 → 問題在更基礎的地方

步驟 2: 檢查瀏覽器控制台（最重要！）
-----------------------------------
1. 按 F12 打開開發者工具
2. 點擊 Console 標籤
3. 查看所有紅色錯誤訊息
4. 複製完整的錯誤訊息

常見錯誤：
- "Cannot find module" → 模組路徑錯誤
- "Unexpected token" → 語法錯誤
- "React is not defined" → React 未導入
- "Failed to compile" → 編譯錯誤

步驟 3: 檢查網路請求
-------------------
1. 在開發者工具中點擊 Network 標籤
2. 重新整理頁面（F5）
3. 查看是否有紅色（失敗）的請求
4. 特別注意 .js 和 .css 檔案

步驟 4: 檢查終端機輸出
-------------------
查看執行 npm run dev 的終端機，是否有錯誤訊息

步驟 5: 提供資訊給我
-------------------
請告訴我：
1. 瀏覽器控制台的完整錯誤訊息
2. 終端機的輸出
3. Network 標籤中失敗的請求

有了這些資訊，我就能準確修復問題！
